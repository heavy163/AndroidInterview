安卓在 IoT 场景下的通信能力，涵盖 BLE 蓝牙、Wi-Fi 及 Matter 智能家居协议。

| 序号 | 技术点 | 核心内容 | 考察权重 |

| 16.2 | [Wi-Fi 通信](./02-WiFi通信/) | Wi-Fi Direct、局域网发现、Socket 通信 | ★★★★☆ |
