# 18 高频面试题汇总

## 概览
30k+ 岗位面试真题汇总，按难度分层：基础 → 架构 → 性能 → 系统与底层。

## 题目分类

| 类别 | 章节 | 典型题目 | 难度 |
|-----|------|---------|:---:|
| 基础题 | [基础面试题](./01-基础面试题/) | Activity 启动模式、Handler 原理、HashMap 扩容 | ★★★ |
| 架构设计题 | [架构设计题](./02-架构设计题/) | 设计一个图片加载框架、IM 消息架构设计 | ★★★★★ |
| 性能优化题 | [性能优化题](./03-性能优化题/) | 启动速度优化全链路、内存泄漏场景分析 | ★★★★★ |
| 系统与底层 | [系统与底层题](./04-系统与底层题/) | AMS 启动流程、Binder 一次拷贝原理 | ★★★★☆ |

## 答题框架

每道题按以下结构组织：
1. **问题复现**：还原面试场景中的真实问法
2. **解题思路**：思维框架与答题方法论
3. **标准答案**：结构化的完整回答
4. **加分项**：能体现深度和广度的扩展内容
5. **常见误区**：面试中容易被追问的薄弱点

> 此模块为考前冲刺用，与各技术方向的六层深度内容形成互补
